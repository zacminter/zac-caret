# Zac^ Documentation

## Your Digital Command Center

Zac^ is a gamified AI agent orchestration platform that transforms project portfolio management into an RTS-inspired experience. Command Claude-powered workers through a 3D game interface, watching your projects evolve from foundations to fully-realized structures.

## Tech Stack

- **Tauri 2.x** - Desktop application shell
- **Bevy 0.14** - 3D game engine (Rust)
- **Leptos** - WebView UI framework (Rust → WASM)
- **SQLite** - Local persistence
- **Claude API** - AI worker brains

---

## V1 Feature Summary

- Zac^ hero unit with autonomous task management
- Worker entities powered by Claude Code CLI
- Project buildings that evolve through 10 visual stages
- Knowledge accumulation system (workers get smarter)
- Token tracking with "food supply" visualization
- Full persistence across sessions
